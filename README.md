# Deep Learning Projects with TensorFlow

This repository contains small deep learning projects built while learning practical implementation of neural networks using Python and TensorFlow.

## Projects

### 1. Customer Churn Prediction (ANN)
Built an Artificial Neural Network (ANN) to predict whether a customer will leave a bank using a customer churn dataset from Kaggle.

### 2. Handwritten Digit Recognition (CNN)
Built a Convolutional Neural Network (CNN) using the MNIST dataset for handwritten digit classification.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset

The dataset used in this project is too large to be uploaded to GitHub.

 - Customer Churn Modeling Dataset (Kaggle)
- MNIST Dataset (Kaggle)
